FACELOCK APP
FaceLock is a secure and user‑friendly facial recognition authentication app. It allows users to unlock devices, 
apps, or sensitive data using their face as the key — combining convenience with cutting‑edge security.

 Features
 A)Fast Authentication – Unlock in seconds using facial recognition.
  B)High Security – Biometric data is encrypted and stored securely.
 C)Cross‑Platform – Works on Android, iOS, and desktop environments.
 D)Fallback Options – PIN or password available if face unlock fails.
  E)Privacy First – No facial data leaves your device.

  INSTALLATION
  CLONE THE REPOSITORY:
  git clone https://github.com/danielwealth/facelock-client.git
cd facelock-client
INSTALL DEPENDENCIES
npm install
RUN THE APP
npm start

USAGE
Open the app.
Register your face by following the on‑screen instructions.
Use FaceLock to authenticate whenever prompted.
